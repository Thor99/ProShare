ProShare is a simple way to discover and share new projects, where you can filter projects based on your search terms or categories! ProShare uses a system of poupularity based in "Fambs" (developed without any gem), so, with more Fambs, more popularity your project will have! The entire site can be used in english or portuguese, you choose! It was made by me, using Ruby on Rails, to increase my rails skills. Have fun :)
